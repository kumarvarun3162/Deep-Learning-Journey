# 🧠 Introduction of Activation Functions in Neural Networks

## 📌 What is an Activation Function?

An **Activation Function** is a mathematical function used in **Neural Networks** to decide **whether a neuron should be activated (fire) or not**.

In simple words, it helps the model **decide what information should move forward and what should be ignored**.

Without activation functions, a neural network would behave like a **simple linear model** and would not be able to learn complex patterns.

---

# 🤔 Why Do We Need Activation Functions?

Activation functions help neural networks:

✅ Learn **complex patterns**
✅ Make **better decisions**
✅ Solve **non-linear problems**
✅ Improve **prediction accuracy**

Without activation functions, a neural network would just perform **simple addition and multiplication**, which is not powerful enough for tasks like:

* Image recognition
* Speech recognition
* Medical diagnosis
* Self-driving cars

# 🧩 How It Works in Neural Networks

A neuron first performs a calculation:

```
Output = (Weights × Inputs) + Bias
```

Then the **activation function** is applied:

```
Final Output = Activation(Output)
```

This final output is then sent to the **next neuron**.

---

Activation functions allow neural networks to:
