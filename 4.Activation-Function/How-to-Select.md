# How to Choose Right Activtion Function

## A few rules for choosing the activation function for thr output layer based on the type of prediction problem 

* **Regression** - Linear Activation Function
* **Binary-Classification** - Sigmoid/Logistic Activation Function
* **Multiclass-Classification** - Softmax Activation Function
* **Multilabel-Classification** - Sigmoid Activation Function
* **CNN** - ReLU Activation Function
* **RNN** - Tanh/Sigmoid Activation Function
