# 🔄 Forward Propagation in Multilayer Perceptron (ANN)

Forward Propagation is the **first step in how a neural network makes predictions**. In this process, the **input data moves forward through the network layer by layer** until the model produces an output.

A neural network usually contains three types of layers:

* **Input Layer** – Receives the input data.
* **Hidden Layer(s)** – Processes the data and learns patterns.
* **Output Layer** – Produces the final prediction.

During forward propagation, the information flows **from the input layer to the output layer**, and no learning happens yet. The network simply calculates the prediction using the current weights and biases.

---

## ⚙️ How Forward Propagation Works

Forward propagation works in a few simple steps.

### 1️⃣ Input Data is Given to the Network

First, the input features are provided to the **input layer**.

For example, if we want to predict whether an email is **spam or not**, the inputs might include:

* Number of suspicious words
* Length of the email
* Presence of links

These inputs are passed to the neurons in the next layer.

---

### 2️⃣ Multiply Inputs with Weights and Add Bias

Each neuron performs a simple mathematical calculation.  
It multiplies the input values with **weights** and then adds a **bias**.

Formula:

```

Z = W · X + b

```

Where:

* **W** = Weight (importance of the input)
* **X** = Input value
* **b** = Bias (extra adjustment)

This step helps the model decide **how important each input feature is**.

---

### 3️⃣ Apply Activation Function

After calculating the value \(Z\), the neuron passes it through an **activation function** such as:

* **ReLU**
* **Sigmoid**
* **Tanh**

The activation function introduces **non-linearity**, which allows the network to learn **complex patterns** in data.

Without activation functions, a neural network would behave like a simple linear model.

---

### 4️⃣ Pass Output to the Next Layer

The output produced by one layer becomes the **input for the next layer**.

This process continues through all hidden layers until the data reaches the **output layer**.

---

### 5️⃣ Final Prediction is Produced

Finally, the **output layer generates the prediction**.

For example:

* Image classification → Cat or Dog
* Email filtering → Spam or Not Spam
* Medical prediction → Cancer or No Cancer

---

## 🧠 Simple Flow of Forward Propagation

```

Input Layer → Hidden Layer(s) → Output Layer
X              H               Y

```

The data simply **moves forward through the network**, which is why it is called **Forward Propagation**.

---

## 🚀 Summary

Forward propagation is the process where **input data flows through the neural network to generate predictions**. Each neuron performs calculations using **weights, bias, and activation functions**, and the result moves to the next layer until the final output is produced.

It is an essential step in training neural networks because it allows the model to **make predictions before calculating errors and updating weights during backpropagation**.
