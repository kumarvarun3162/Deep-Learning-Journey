# 🔙 Backpropagation in Multilayer Perceptron (ANN)

Backpropagation is the **learning process of a neural network**. After the model makes a prediction using forward propagation, backpropagation helps the network **correct its mistakes by adjusting the weights and biases**.

In simple words, backpropagation tells the model **how wrong its prediction was and how to improve it**.

The error is calculated at the **output layer** and then sent **backward through the network** so that each neuron can update its weights.

---

## ⚙️ How Backpropagation Works

Backpropagation works in the following simple steps.

### 1️⃣ Calculate the Error (Loss)

After forward propagation, the model produces a prediction.  
This prediction is compared with the **actual correct value** using a **loss function**.

Example loss functions:

* **Mean Squared Error (MSE)** – used in regression problems
* **Cross-Entropy Loss** – used in classification problems

The loss tells us **how far the prediction is from the correct answer**.

---

### 2️⃣ Send the Error Backward

Once the error is calculated, it is sent **backward from the output layer to the hidden layers**.

This step helps determine **which neurons contributed the most to the error**.

---

### 3️⃣ Calculate Gradients

The network calculates **gradients**, which tell us **how much each weight should change** to reduce the error.

Gradients are calculated using the **chain rule from calculus**.

This step helps the model understand:

* Which weights are too large
* Which weights are too small

---

### 4️⃣ Update the Weights

Finally, the weights are updated using an **optimization algorithm** such as **Gradient Descent**.

Weight update formula:

```

New Weight = Old Weight − (Learning Rate × Gradient)

```

Where:

* **Learning Rate** – controls how big the update step is
* **Gradient** – direction to reduce the error

By repeating this process many times, the model gradually **reduces the error and improves its predictions**.

---

## 🧠 Simple Flow of Backpropagation

```

Forward Pass:   Input → Hidden Layer → Output → Prediction

Backward Pass:  Error ← Hidden Layer ← Output

```

The error moves **backward through the network**, which is why it is called **Backpropagation**.

---

## 🚀 Summary

Backpropagation is the process that allows a neural network to **learn from its mistakes**. It calculates the error between the predicted and actual output, sends this error backward through the network, and updates the weights to reduce the error.

By repeating forward propagation and backpropagation many times during training, the neural network becomes **better at making accurate predictions**.
